# EuMacaronesiaAnalysis
CMEMS Marine variables Data Analysis in European Macaronesia

# Macaronesia Oceanographic Analysis: Spatial Variability, Long-term Trends, and Vertical Structure

This repository contains the Python notebooks used for the oceanographic data analysis in the paper *"Decadal Patterns and Vertical Gradients of Oceanographic Variability
in European Macaronesia: A Multi-Parameter Satellite and Reanalysis
Synthesis"* (submitted to Journal of Marine Systems).

## 📋 Description

The notebooks perform the analyses described in the paper.

## 🚀 Prerequisites

- Python 3.10+
- Jupyter Notebook / JupyterLab
- CMEMS account for data download (optional, data is included in the `data` folder)
